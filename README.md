# Analyzing Historical Stock & Revenue Data and Building a Dashboard

## Project Overview
This project analyzes historical stock prices and revenue data for companies such as **Tesla** and **GameStop**.  
The goal is to extract, clean, analyze, and visualize the data using Python and build an interactive dashboard.

## Tools & Technologies
- Python
- Pandas
- NumPy
- yfinance
- BeautifulSoup
- Matplotlib / Plotly
- Power BI / Tableau

## Project Steps
1. Extract historical stock price data using yfinance
2. Scrape revenue data from public sources
3. Clean and preprocess datasets
4. Perform exploratory data analysis
5. Build an interactive dashboard
6. Present insights visually

## Key Insights
- Stock price trends vs company revenue
- Growth and volatility comparison
- Revenue impact on stock performance
